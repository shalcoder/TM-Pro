# 🎬 TM PRO - Video Demo Script

Follow this script to create a professional 2-minute demonstration of your system.

## Scene 1: The First Impression (0:00 - 0:20)
- **Visual**: Login Screen.
- **Action**: Use Google SSO or create a new account.
- **Script**: "Welcome to TM PRO, an enterprise-grade multi-tenant workspace. Notice the premium dark-mode interface designed for high-performance teams."

## Scene 2: The Command Center (0:20 - 0:50)
- **Visual**: Main Dashboard.
- **Action**: Create 2-3 tasks with different priorities. Move one to 'In Progress'.
- **Script**: "Our Kanban board supports a 5-stage engineering lifecycle. Here, I'm deploying a critical operation. The system automatically calculates our real-time velocity in the summary bar above."

## Scene 3: The Insights Engine (0:50 - 1:20)
- **Visual**: Analytics Tab.
- **Action**: Hover over the Area Chart. Scroll through the Activity Feed.
- **Script**: "The Analytics engine provides deep visibility. We can track team efficiency, burndown depth, and monitor a live stream of organizational activity—all strictly isolated to this tenant."

## Scene 4: Security & Accountability (1:20 - 1:45)
- **Visual**: Audit Trail (Admin Only).
- **Action**: Show the table of logs.
- **Script**: "Security is at our core. Every action is recorded in our persistent Audit Trail. We store JSON snapshots of every change, allowing admins to see exactly what was modified and by whom."

## Scene 5: Strict Multi-Tenancy (1:45 - 2:10)
- **Visual**: Log out and log in to a different Org.
- **Action**: Show an empty dashboard.
- **Script**: "Finally, we demonstrate strict data isolation. By switching to a different organization, you can see that zero data leaks across boundaries. The entire environment is fully containerized using Docker and PostgreSQL."
